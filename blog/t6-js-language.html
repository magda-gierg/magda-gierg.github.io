use cases for parentheses, braces, brackets and quotes in JavaScript language

parentheses ( )

  1. Execute a function e.g.var result = doSomething()
  2. Provide arguments for function declaration e.g.function doSomething (how, when) {}
  3. Provide conditional statement condition for loop, if, and switch statements e.g.while (var < 0) {do something}
  4. Execute function with parameters e.g.functionName(parameter1, parameter2, parameter3) {}
  5. Surround conditional statements e.g.
    if (condition) {
    >statement_1;
    } else {
    statement_2;
    }
  6. Provide switch statement conditions e.g.
    switch (fruittype) {
    case 'Oranges':
    console.log('Oranges are $0.50 a pound.');
    break;
    case 'Apples':
    console.log('Apples are $0.30 a pound.');
    break;
    default:
    console.log('Sorry, we are out of ' + fruittype + '.');
    }
    console.log("Is there anything else you'd like?");

brackets [ ]

  1. Creating arrays e.g.myArray = ["England", "Germany", "Poland"];
  2. Used for accessing the properties of an Object (or the elements of an Array) e.g.
    myArray = ["England", "Germany", "Poland"];
    console.log(myArray[2]);

braces { }

  1. Creating objects e.g.var car = {type:"Fiat", model:"500", color:"white"};
  2. Creating conditional statements e.g.
    if (hour < 18) {
    greeting = "Good day";
    }
  3. Function declaration e.g.
    function functionName (parameters) {
    code to be executed
    }

single quotes ' '

  1. Creating string e.g.var hello = "Hello, how are you?";
  2. Accessing object properties e.g.objectName['propertyName']

double quotes " "

  1. Like single quotes creating string e.g.var hello = "Hello, how are you?";
  2. Like single quotes accessing object properties e.g.objectName["propertyName"]
  3. Creating object properties e.g.var person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"};
