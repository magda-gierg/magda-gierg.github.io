js language

use cases for parentheses, braces, brackets and quotes in JavaScript language


parentheses ( )

to execute a function:
var result = doSomething()

to provide arguments for function declaration:
function doSomething (how, when) {}

to provide conditions in loops:
for (var < 0) {do something}

to execute function with parameters:
functionName(parameter1, parameter2, parameter3) {}

to surround conditional statements:
if (condition) {
>statement_1;
} else {
statement_2;
}

to create switch statement conditions:
switch (fruittype) {
case 'Oranges':
console.log('Oranges are $0.50 a pound.');
break;
case 'Apples':
console.log('Apples are $0.30 a pound.');
break;
default:
console.log('Sorry, we are out of ' + fruittype + '.');
}
console.log("Is there anything else you'd like?");

brackets [ ]

to create arrays:
myArray = ["red", "blue", "yellow"];

to access array elements by their index:
myArray = ["red", "blue", "yellow"];
console.log(myArray[2]);

braces { }

to create objects:
var car = {type:"Fiat", model:"500", color:"white"};

to create conditional statements:
if (hour < 18) {
greeting = "Good day";
}

to initialize a function:
function start () {init part};

single quotes ' ' and double quotes " "

both can be used to create a string:
var hello = 'Hello, how are you?';
var hello = "Hello, how are you?";

both can be used to access object properties:
objectName['propertyName']
objectName["propertyName"]

but double quotes has to be used to create object properties:
var person = {firstName:"John", lastName:"Doe"};
